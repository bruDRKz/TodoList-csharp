ToDo List
Este repositório contém um ToDo List simples, desenvolvido como parte de um exercício pessoal para aplicar conceitos e práticas da linguagem C# e do desenvolvimento com o .NET. 
Inicialmente, o projeto foi criado para armazenar e organizar tarefas de forma prática, mas ao torná-lo público, decidi realizar algumas melhorias e refatorações no código.

Objetivo
O objetivo principal do projeto é colocar em prática o conhecimento adquirido em C#, .NET, Windows Forms e outras tecnologias relacionadas. 
Algumas áreas do código ainda podem ser aprimoradas para garantir uma implementação mais eficiente e aplicável em cenários mais complexos, como:

Melhoria na estrutura do código.
Refatoração para aplicar padrões de design mais adequados.
Adição de novos recursos e funcionalidades.
Status
Este projeto ainda está em desenvolvimento e algumas melhorias estão sendo feitas, principalmente para aplicar conceitos de boas práticas de desenvolvimento, refatoração e testes unitários.

Tecnologias utilizadas
C#: Linguagem de programação.
.NET Framework: Plataforma de desenvolvimento.
Windows Forms: Para construção da interface gráfica.
SQL Server: Para armazenamento dos dados.
